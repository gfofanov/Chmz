высылаю бэкап и описание БД
 
хранимые процедуры
 
getLastListByDate
- возвращает последние измерения по всем параметрам на дату-время последнюю до указанной даты. Сортировка по кодам точек
 
входной параметр
      @LastDate - дата-время на которую выдаются сведения
выходная таблица
     FacilityCode  - код точки
     ParamName - название параметра
     Value          - значение измерения
     UnitName    - единица измерения
     MeasDT       - дата-время измерения
     Color          - цвет раскраски (0 - серый,  1- красный, 2 - желтый, 3 - зеленый)
 
getParamByPeriod
- возвращает список измерений параметра (код точки равен @FacilityCode) за период с @FirstDate по @LastDate. отсортирован по дате-времени
 
входные параметры
     @FacilityCode  - код точки (для задания параметра)
     @FirstDate      - начальная дата-время интервала
     @LastDate      - конечная дата-время интервала
выходная таблица
     Value          - значение измерения
     UnitName    - единица измерения
     MeasDT       - дата-время измерения
     Color          - цвет раскраски (0 - серый,  1- красный, 2 - желтый, 3 - зеленый)
 
 
Восстановление из резервной копии
 
1. Заходим в SQL Server Managment Studio
2. В "обозревателе объектов" выбираем "базы данных"
3. по контекстному меню выбираем "Восстановить базу данных"
4. В окне задаем:
       имя БД - "iac54o2sk"
       источник "С устройства" и указываем откуда берем файл бэкапа
